[Fileck](https://github.com/SandmeyerX/fileck) 是一款使用Rust编写的高性能、容易上手的文件校验工具，提供以下有用的功能：

-   批量校验多个文件
-   指定校验算法（具体请查看[支持的校验算法](#algorithms)）
